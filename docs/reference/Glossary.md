# Glossary

| Term | Meaning |
|------|---------|
| **SkillATS** | The hiring product at skillats.com |
| **Dashboard** | Home screen after you sign in |
| **Job** | An open role you hire for |
| **Board / pipeline** | Stages (columns) for a job’s candidates |
| **Stage** | A step in hiring, e.g. Applied or Interview |
| **Candidate** | A person in your talent pool |
| **Career site** | Public pages where people browse jobs and apply |
| **AI Candidate Search** | Chat that finds matching people already in SkillATS |
| **AAA** | AI Assignment Analyzer — forward broker emails, get shortlists |
| **Prompt (on AAA)** | Instructions you write once for every assignment |
| **Shortlist** | Top matching candidates (AAA sends up to 10) |
| **Platsbanken** | Swedish public job board you can publish to |
| **Integration** | Connection to another hiring tool |
| **GDPR / privacy** | Consent and rules for storing candidate data |
